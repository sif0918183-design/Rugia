# تطبيق الرقية الشرعية PWA

تطبيق ويب متقدم للرقية الشرعية يعمل كتطبيق PWA (تطبيق ويب تقدمي).

## المميزات

- ✅ تشغيل تلقائي للمقاطع الصوتية
- ✅ تكرار القائمة عند الانتهاء
- ✅ واجهة عربية احترافية
- ✅ يدعم وضع عدم الاتصال
- ✅ قابل للتثبيت على الجهاز
- ✅ متوافق مع جميع الأجهزة

## التثبيت والتشغيل

### الطريقة الأولى: Vercel (مستضافة)
1. انسخ رابط GitHub لهذا المشروع
2. اذهب إلى [vercel.com](https://vercel.com)
3. أنشئ حساب جديد أو سجل الدخول
4. اضغط على "New Project"
5. اختر المستودع (Repository) الخاص بك
6. اضغط على Deploy
